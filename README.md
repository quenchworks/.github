# `.github`

Organization-level defaults and community health files for
[Quenchworks](https://github.com/quenchworks), a catalog of hardened, 0-CVE Helm charts and
container images built from source.

The contents here apply across every repo in the org unless a repo overrides them.

| Path | What it is |
|------|------------|
| [`profile/README.md`](./profile/README.md) | The org landing page shown at [github.com/quenchworks](https://github.com/quenchworks) |
| [`profile/assets/`](./profile/assets/) | Logo and icon (light/dark) used by the profile |
| [`SECURITY.md`](./SECURITY.md) | Disclosure policy and how to verify signatures/SBOMs |
| [`CONTRIBUTING.md`](./CONTRIBUTING.md) | How to propose an app, file issues, and open PRs |
| [`CODE_OF_CONDUCT.md`](./CODE_OF_CONDUCT.md) | Contributor Covenant |
| [`ISSUE_TEMPLATE/`](./ISSUE_TEMPLATE/) | Issue forms (org defaults) |

> Brand assets are the iso-layered-cube mark in monochrome. Don't recolor or alter proportions.
